Adding Columns to a DataFrame

There are two main methods to add columns to a DataFrame in Pandas:

1. Direct Method

Add a column without specifying a position. The column will be added at the end.

# Creating a DataFrame import pandas as pd data = { "name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "department": ['tech', 'technical', 'technologya', 'fighting', 'relation', 'sckills'], "salary": [80000, 70000, 40000, 30000, 49000, 91000] } df = pd.DataFrame(data) print("Original DataFrame:") print(df) print("\n") # Adding a column using direct method df["bonus"] = df['salary'] * 0.1 print("DataFrame with bonus column:") print(df)

2. Insert Method

Add a column at a specific position using the insert() method.

# Creating a DataFrame import pandas as pd data = { "name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "department": ['tech', 'technical', 'technologya', 'fighting', 'relation', 'sckills'], "salary": [80000, 70000, 40000, 30000, 49000, 91000] } df = pd.DataFrame(data) print("Original DataFrame:") print(df) print("\n") # Adding a column using insert method df.insert(2, "Age", [23, 45, 34, 25, 36, 29]) print("DataFrame with Age column:") print(df)

Updating Values in a DataFrame

Pandas provides several ways to update values in a DataFrame. The most common method is using .loc[] to access specific locations.

Updating a Single Value

Use .loc[row_index, column_name] to update a specific value.

# Creating a DataFrame import pandas as pd data = { "name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "department": ['tech', 'technical', 'technologya', 'fighting', 'relation', 'sckills'], "salary": [80000, 70000, 40000, 30000, 49000, 91000] } df = pd.DataFrame(data) print("Original DataFrame:") print(df) print("\n") # Updating a single value df.loc[3, "salary"] = 35000 print("DataFrame after updating salary at index 3:") print(df)

Updating Multiple Values

Use .loc[:, column_name] to update all values in a column.

# Creating a DataFrame import pandas as pd data = { "name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "department": ['tech', 'technical', 'technologya', 'fighting', 'relation', 'sckills'], "salary": [80000, 70000, 40000, 30000, 49000, 91000] } df = pd.DataFrame(data) print("Original DataFrame:") print(df) print("\n") # Updating all values in a column df.loc[:, "salary"] = df["salary"] + 1000 print("DataFrame after increasing all salaries by 1000:") print(df)

Removing Columns from a DataFrame

To remove columns from a DataFrame, use the drop() method with the columns parameter.

Syntax

# Basic syntax for removing columns df.drop(columns=["column_name"], inplace=True)

The inplace=True parameter modifies the DataFrame directly without creating a copy.

Example

# Creating a DataFrame import pandas as pd data = { "name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "department": ['tech', 'technical', 'technologya', 'fighting', 'relation', 'sckills'], "salary": [80000, 70000, 40000, 30000, 49000, 91000] } df = pd.DataFrame(data) print("Original DataFrame:") print(df) print("\n") # Removing a single column df.drop(columns=["department"], inplace=True) print("DataFrame after removing department column:") print(df)

Removing Multiple Columns

To remove multiple columns, pass a list of column names to the columns parameter.

# Creating a DataFrame import pandas as pd data = { "name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "department": ['tech', 'technical', 'technologya', 'fighting', 'relation', 'sckills'], "salary": [80000, 70000, 40000, 30000, 49000, 91000] } df = pd.DataFrame(data) print("Original DataFrame:") print(df) print("\n") # Removing multiple columns df.drop(columns=["department", "salary"], inplace=True) print("DataFrame after removing department and salary columns:") print(df)

Handling Missing Values in Pandas

Missing values are common in real-world datasets. Pandas provides several methods to detect and handle them.

1. Detecting Missing Values

Use isnull().sum() to count the number of missing values in each column.

3. Aggregation Functions

Pandas provides various aggregation functions to summarize data:

  • sum(): Sum of values
  • mean(): Arithmetic mean
  • median(): Median (50th percentile)
  • min(): Minimum value
  • max(): Maximum value
  • count(): Number of non-NA values
  • std(): Standard deviation
  • var(): Variance

Example: Using Aggregation

Merging and Joining DataFrames

Merging and joining are operations that combine DataFrames based on common columns or indices.

Syntax

# Basic syntax for merging DataFrames pd.merge(df1, df2, on="column_name", how="join_type")

Parameters:

  • df1, df2: DataFrames to merge
  • on: Column name to join on
  • how: Type of join ('inner', 'outer', 'left', 'right')

Types of Joins

  • Inner Join: Returns only rows where key values exist in both DataFrames
  • Outer Join: Returns all rows from both DataFrames, filling missing matches with NaN
  • Left Join: Returns all rows from the left DataFrame, and matching rows from the right
  • Right Join: Returns all rows from the right DataFrame, and matching rows from the left

Example

# Creating two DataFrames import pandas as pd Customer_data = { "Name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "customer_id": [1, 2, 3, 4, 5, 6] } df1 = pd.DataFrame(Customer_data) Order_data = { "price": [500, 700, 400, 300, 490, 910], "customer_id": [1, 2, 3, 4, 5, 8] } df2 = pd.DataFrame(Order_data) print("Customer DataFrame:") print(df1) print("\nOrder DataFrame:") print(df2) print("\n") # Inner join print("Inner Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="inner") print(merged_data) print("\n") # Outer join print("Outer Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="outer") print(merged_data) print("\n") # Left join print("Left Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="left") print(merged_data) print("\n") # Right join print("Right Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="right") print(merged_data)

Concatenating DataFrames

Concatenation is the process of joining DataFrames either vertically (row-wise) or horizontally (column-wise).

Syntax

# Basic syntax for concatenating DataFrames pd.concat([df1, df2], axis=0/1, ignore_index=True)

Parameters:

  • [df1, df2]: List of DataFrames to concatenate
  • axis=0: Concatenate vertically (row-wise)
  • axis=1: Concatenate horizontally (column-wise)
  • ignore_index=True: Reset the index of the resulting DataFrame
